# Sản phẩm Tekmonk

Game's name: "Thánh Gióng cứu nước".

Idea to make the game is the mythology story "Thánh Gióng".

## requirements:

To play the game, visit the link below:
[Views](https://scratch.mit.edu/projects/1273148291/)

## Installation:

To install the game to your computer, follow the instructions:

1. Download it
2. Open scratch 
3. Click on the folder button on the top left corner of your screen
4. Then click the load from the computer button
5. select the game

## How to use:

Game play:
- "w" to jump.
- Bamboo = invincible for 4 seconds.
- Rice bowl for 1 health

Menu:
- Start button to play.
- Instructions button to know how to play.
- Sound button to turn on/off gameplay sounds.
